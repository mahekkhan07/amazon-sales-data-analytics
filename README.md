# Amazon Sales Data Analytics

## Project Overview
This project analyzes Amazon Sales data to identify revenue trends, customer behavior, fulfillment performance, and product demand patterns. The goal is to generate actionable business insights using Python and data visualization techniques.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Exploratory Data Analysis (EDA)

## Dataset
- 128,975 records
- 24 features
- Key metric: Order Amount (INR)

Dataset Link: [Download Dataset](https://drive.google.com/file/d/1Q-j51LXX08x8BfvhQgm4Zdk85gk27PWm/view?usp=drivesdk)

Note: The dataset is hosted externally due to GitHub's 25MB file size limit.

## Key Insights
- Major revenue contribution comes from metro cities.
- Retail (B2C) customers dominate overall sales.
- Amazon-fulfilled orders perform better than merchant fulfillment.
- Certain product categories generate higher revenue.
- Standard shipping is the most commonly used method.

## Project Outcome
Performed data cleaning, analysis, and visualization to extract meaningful business insights that can support sales strategy and operational decision-making.
